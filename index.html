Some Projects Ive Worked on

LEAD

LEAD is an Agile Project Management web application developed in Angular utilizing custom REST APIs on the backend to handle data. The backend APIs are developed in PHP using Laravel as well as NodeJS utilizing ExpressJS. Each API service uses its own database, including MySQL and MongoDB databases, to adhere to best microservice practices.

SuppSupply

SuppSupply is a supplement Ecomomerce web application developed in Angular with an ExpressJS API running on the backend. The ExpressAPI uses a data access object facade to connect to a MySQL database. Ngrx state management is incorporated to reduce HTTP requests and simplify the frontend development experience.
